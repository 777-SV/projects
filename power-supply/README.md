Hi-Fi PSU

Контекст: Частина системи 4-канального передпідсилювача з МК-керуванням.

Технічні особливості:

Схема Dual Mono (2 окремих силових трансформатори).

Окремий черговий трансформатор для логіки (Standby режим).

Проектування: NI Multisim (симуляція) та DipTrace (PCB).

Корпус: розроблено в SolidWorks, деталі для STL друку.
